# Sprint 1 — Homepage update

This update replaces only one file in your existing project:

```text
src/pages/index.astro
```

## Apply the update

1. Download and extract this ZIP file.
2. Open the extracted `src/pages` folder.
3. Copy `index.astro`.
4. Paste it into your project folder:

   ```text
   D:\Shubhek\Website\Projects\src\pages
   ```

5. Choose **Replace the file in the destination** if Windows asks.
6. Your local website should refresh automatically. If it does not, refresh `http://localhost:4321` in the browser.

No `npm install` is required for this update.
